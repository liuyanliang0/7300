5.3.类的成员

- 字段，存储类的基本信息
- 属性，获取或设置类的基本信息
- 方法，我们让“类”做一些事情、干一些活儿，“事情和活儿”就是方法
- 构造函数，通过构造函数，我们能在程序中“制造”出一个类
- 析构函数，释放类所占用的系统资源
- 索引器，类似属性，但更加专业（简单了解，不展开）
